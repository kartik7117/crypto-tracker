# Crypto Tracker

A simple **Cryptocurrency Tracker Web Application** built with **Spring Boot** that allows users to search and view cryptocurrency price data.

## Features
- Search for cryptocurrencies
- View real-time crypto price data
- Simple and clean UI
- Uses external crypto API to fetch market data

## Tech Stack
- Java
- Spring Boot
- REST API
- Thymeleaf / HTML
- Maven

src
  ├── controller
  ├── service
